  └─ http://127.0.0.1:8000/dashboard/ → Redirects to admin dashboard
  └─ http://127.0.0.1:8000/admin-dashboard/ → Admin Dashboard
  └─ http://127.0.0.1:8000/allocate/ → Allocate Rooms

  └─ http://127.0.0.1:8000/admin/ → Django Admin
  └─ [Logout] → Redirects to login

STUDENT AFTER LOGIN:
  └─ http://127.0.0.1:8000/dashboard/ → Redirects to student dashboard

  └─ http://127.0.0.1:8000/pay/ → Make Payment

  └─ http://127.0.0.1:8000/complaints/ → View Complaints

  └─ [Logout] → Redirects to login
