## 🎓 How It Works

### Admin Workflow
1. Admin logs in → Redirected to admin dashboard
2. Sees: Rooms (5), Students (5), Complaints (3), Revenue (₹5,000)
3. Can: Allocate rooms, view/respond to complaints
4. Analytics: Statistics updated automatically

### Student Workflow
1. Student logs in → Redirected to student dashboard
2. Sees: Profile, fees (due/paid/outstanding), room assignment
3. Can: Pay fees (recorded to DB), submit complaints
4. Payment recorded: Database updated instantly

### Payment Recording
```
Student Pays ₹500:
  1. Payment form submitted
  2. Amount validated (Decimal precision)
  3. Payment record created in database
  4. StudentProfile.fees_paid updated
  5. Outstanding recalculated
  6. Success message shown
  7. Redirected to dashboard
```
