## 🧬 DATABASE

### How Payment Recording Works

**Student clicks "Pay Fees"**
```
1. Form submitted with amount and method
2. Amount validated (must be > 0)
3. Decimal created from input
4. Payment record inserted into database
5. StudentProfile.fees_paid updated
6. Outstanding recalculated automatically
7. Success message shown
8. Redirect to dashboard
```

**What's Saved in Database**
```
Payment Table:
- student_id: 2 (John Doe)
- amount: 500.00
- paid_on: 2026-01-24 10:30:45
- method: online

StudentProfile Update:
- fees_paid: 5000.00 → 5500.00
- outstanding: Recalculated
```
